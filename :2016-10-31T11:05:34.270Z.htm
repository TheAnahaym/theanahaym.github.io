Установка Windows XP на Hyper-V 2016 возможна. Для правильной работы необходимо установить пакет интеграции от Windows Server 2008 R.
 An installation of Windows XP on Hyper-V 2016 is possible. For properly work it is necessary to install the integration package from Windows Server 2008 R2.
 Die Installierung von Windows XP auf Hyper-V 2016 ist möglich. Für die korrekte Arbeit ist es notwendig das Integrationspaket von Windows Server 2008 R2 zu installieren.
