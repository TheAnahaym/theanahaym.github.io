Редко приходится обновлять ядро для Linux, т.к. обычно ядро по умолчанию удовлетворяет всем требованиям и всем приложениям. Однако, в процессе установки MS SQL 2016 на Debain 8.6 c ядром 3.16 выяснилось, что данное ядро по умолчанию не пригодно для установки MS SQL 2016. Для успешной установки необходимо обновить OpenSSL. Позже выяснил, что Ubuntu 16.04, на которую можно официально установить MS SQL 2016, имеет ядро 4.4. По этому я и задался вопросом обновления ядра в Debian 8.6. Делается это за 5 минут:

uname -mrs
Linux 3.16.0-4-amd64 x86_64

echo "deb http://ftp.de.debian.org/debian jessie-backports main" >> /etc/apt/sources.list
aptitude update
aptitude search linux-image
aptitude install -t jessie-backports linux-image-amd64
aptitude clean
reboot

uname -mrs
Linux 4.7.0-0.bpo.1-amd64 x86_64

 После этого установка MS SQL выполняется за пару команд.
