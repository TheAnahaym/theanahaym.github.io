Как и любая ОС, Linux начинается с установки. На своих серверах я использую дистрибутив Debian. Далее, немного о простой установке Debian. Образ дистрибутива можно скачать здесь. Для установки достаточно скачать только первый CD. Установка производилась на виртуальную машину Hyper-V 2012: CPU i7-2600, 2 GB RAM. Внимание: сетевая карта должны быть Legacy! Мы будем рассматривать установку без графического интерфейса. Так же, мы не будем устанавливать графическую оболочку, т.к. установка будет произведена для сервера. Однако, есть случаи, когда приложению может потребоваться графическая оболочка. Для статьи используется дистрибутив 6.0.6 х64

Выбор языка установки:

Выбор местоположения. От этого зависит часовой пояс и время на сервере.

Выбор кодировки. Т.к. болшенство программ на английском языке, я устанавливаю американскую кодировку UTF-8

Выбор раскладки клавиатуры.

Если в вашей сети нет DHCP сервера, вам нужно вручную настроить сетевую карту. Перед установкой в свойствах виртуальной машины необходимо указать Legacy сетевую карту, т.к. с синтетической сетевой картой ядро 2.6 не будет работать. После обновления ядра до весрии 3.4 Можно будет использовать синтетическую карту Hyper-V.

Задаём имя сервера

Т.к. у меня используется DHCP сервер он по умолчанию указал домен. Если у вас нет домена - пишем сюда рабочую группу.

Задаём и повторяем пароль для суперпользователя root.

Создаём пользователя без административных полномочий. Задаём и повторяем пароль для него.

Выбираем метод использования диска. Я использую по-умочланию. Немного об LVM.

Выбираем какие и сколько у нас будет разделов. По аналогии с Windows - логических дисков. Для тестовых серверов можно работать с одним разделом.

Так как я устанавливаю только систему без приложений я не буду сканировать другие диски на предмет дополнительных пакетов.

Некоторые пакеты на дисках могут быть устаревшими. Последние версии можно получить по сети.

Выбираем сервер поближе. У меня это немецкие сервера.

Указываем прокси сервер, если такой используется в огранизации для доступа к интернету.

Отказываюсь от установки "популярного контента"

Отключаю (ПРОБЕЛОМ) установку графической оболочки и добавляю SSH server для доступа и сети.

Устанавливаем загрузчик ОС GRUB. Он умеет загружать не только Linux, но и Windows.

Сразу после установки я, как правило, устанавливаю сервер NTP для синхронизации времени через интернет atp-get install ntp. На этом - всё.
