• Восстановление Active Directory 2012
    • Контроллер домена 2012
    • Контроллер домена 2012 и NAT
    • Обновление Active Directory 2008R2 до 2012 
    • Резервное копирование Active Directory 2012 
    • Доверительные отношения между 2008 R2 и 2012

 

 

  Приступим к восстановлению Active Directory 2012 из резервной копии System State. Восстановление будет происходит из режима Directory Services Repair Mode. Он становится доступным после установки ролей Active Directory и настройки домена. Для входа в данный режим необходимо перезагрузить сервер и нажать F8.

Статья не завершена, т.к. восстановление хочу произвести на другом сервере. В этом случае встретил некоторые нюансы.
