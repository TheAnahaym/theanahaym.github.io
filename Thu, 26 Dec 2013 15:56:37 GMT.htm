 На работе появился новый проект, но его тестирование не представлялось возможным на нынешнем компьютере: не хватало производительности дисковой системы для виртуальных машин. По этому, после общения с руководством, было решено обновить системному администрутору, то есть мне, его рабочий компьютер. И я решил воспользоваться новыми комплектующими для написания статей (давно пора восполнить этот пробел) об устройстве, сборке компьютера, а так же аглоритма решения неисправностей компьютера. В небольших компаниях эта функция так же выполняется системным администратором. В крупных организациях, как правило, этим занимается специалист из отдела Технической Поддержки. Некоторые комплектующие из обзора в дальнейшем не будут использоваться в моём рабочем компьютере. Выбранные комплектующие не являются рекомендацией! Сразу хочу извиниться за качество фотографий - быстро стемнело... И так, необходимый набор комплектующий для компьютера (слева на право):

 - Блок Питания (БП)
 - Sata кабель
 - Жёсткий диск (HDD)
 - Твёрдотельный жёсткий диск (SSD)
 - Корпус
 - Материнская Плата (МП)
 - Видеокарта
 - Процессор
 - Оперативная память (на фотографии отсуствует)

 Цена за набор без SSD, видеокарты и оперативной памяти, но с четырьмя HDD - 720 евро. Оперативная память будет использоваться от старого компьютера. Видеокарта встроена в процессор.

 Начну обзор с Корпуса, так как он всегда на виду у пользователя.  Он же системный блок, но для меня без комплектующий он просто корпус. Он служит для установки комплектующих, защиты от воздействия внешних факторов. Критерии, по которым я выбираю Корпус для компьютера:
 - форм-фактор
 - удобство доступа к комплектующим во время технического обслуживания компьютера
 - возможность установки необходимого количества комплектующих
 - удобство подключения внешних USB-устройств и наушников
 - проветриваемость

 Подробнее о наиболее распространённом форм-факторе ATX. Ниже изображён Корпус AeroCool Strike-X и его комплектующие: болты и стяжки, подставки для дисков, запасная передняя панель для флоппи-дисковода или кардридера. Ещё есть документация к корпусу, но не на фоторгафии. Не очень удобное монтирование жёстких дисков, но устанавливаемое их количество невелировало этот недостаток. Хорошая проветриваемость для охлаждения системы. Я пока не встречал корпус, способный защитить систему от накапливания пыли, если пол помещения просто не моется.

 Системная плата или более известная как Материнская плата - служит для крепления комплектующих и обеспечения связи между ними. Мои критерии выбора материнской платы:
 - сокет процессора
 - поддержка встроенного в процессор видео
 - количество модулей для подключения оперативной памяти
 - количество модулей для подключения PCI устройств
 - тип и количество подключаемых жёстких дисков
 - сайт технической поддержки для скачивания обновлений BIOS и драйверов

 Форм-фактор опредлеяется количеством подключающихся копмлектующих. Так уж сложилось, что я являюсь поклонником материнских плат фирмы Gigabyte, по этому в первую очередь рассматривал материнские платы это фирмы. Следующую фирму, которую я бы рассмотрел - ASUS. Избегаю использования материнских плат компании Foxconn, которые они выпускают под своим брендом, хотя делают много чего для других компаний. Ниже изображён комплект поставки: материнская плата, две документации, заглушка для корпуса, пара SATA кабелей и диск с драйверами.

 Обозначение наиболее важных компонентов материнской платы:
 0 - батарейка для BIOS
 1 - гнездо для установки процессора
 2 - подключение питания для процессора
 3 - подключение куллера процессора
 4 - подключение питания материнской платы
 5 - слоты для установки оперативной памяти
 6 - слоты для установки PCI-E1 устройств
 7 - слот PCI-E16 версии 3 для установки видеокарты
 8 - чип Z87 с пассивным охлаждением
 9 - контроллер моста PCI в PCI-E
 10 - чип ITE - контроллер LPT, COM, PS/2
 11 - слоты PCI
 12 - разъёмы SATA III для подключения жёстких дисков, SSD или оптических приводов. Совместимы с SATA и SATA II
 13 - подлючение аудио на передней панели корпуса
 14 - подключение USB 3.0 на передней панели корпуса
 15 - подключения кнопок питания\сброса и индикаторов на корпусе

 16 - USB 2.0
 17 - PS/2 для подключения клавиатуры или мышки по PS/2
 18 - D-SUB 15-контактный разъём для подключения монитора по стандарту VGA. Использует Аналоговый сигнал
 19 - DVI-D разъём для подключения монитора. Использует только цифровой сигнал
 20 - HDMI разъём для подключения монитора. Использует цифровой сигнал с передачей аудио
 21 - USB 3.0
 22 - Гигабитный Ethernet порт для подключения к ЛВС
 23 - порты для подключения аудио аппаратуры 5.1

 
 Процессор, он же "камень", он же "мозг". Критерии выбора процессора:
 - поддержка встроенного видео
 - поддержка технологии виртуализации
 - тактовая частота процессора и шины
 - кэш процессора
 - цена (можно опустить, так как оплачивается компанией)

 Ниже изображён комплект поставки BOX: с инструкцией и куллером для охлаждения.

 Вот, нашёл на складе старенькие процессоры AMD для сравнения. Так получилось, что 95 % опыта работы с процессорами пришлось на процессоры компании INTEL. Так же изображена термопаста, необходимая для обеспечивания лучшей теплопередачи от процессора к куллеру. И сравнения обратной стороны процессоров. Благодаря "ключам", процессоры невозможно установить в сокет непредназначенный для данного процессора.

 Жёсткий диск - предназначен для храненния информации. Энергонезависимая память. Подробная информация об устройстве диска. Критерии выбора дисков HDD (Hard Disk Drive):
 - объём
 - скорость вращения пластин
 - тип подключения

 Критерии выбора дисков SSD (Solid-State-Drive):
 - объём
 - цена
 - производитель
 - отзовы и рекомендации

 Ниже приведены 5 разных дисков и типы их подключения.
 1 - SATA диск
 2 - SCSI диск
 3 - SSD SATA диск
 4 - SAS диск
 5 - IDE диск

 Для своего рабочего компьютера в качестве системного диска я буду использовать обычный SATA III диск. Четыре новых SATA III диска будут использованы для создания RAID-10 (или средствами операционной системы или средствами материнской платы) для образов виртуальных машин.

 Небольшое отступление по поводу терминологии. Не знаю как получилось, но не так давно я обнаружил, что в качестве обозначения диска стало использоваться слово "накопитель". До этого в магазинах продавались сетевые хранилища, а сейчас - сетевые накопители. Я не могу принять это слово. Не могу сказать - "накопи файл на диск Ц", а вот "сохрани файл на диск Ц" совсем другое дело. Для меня накопитель это что-то вроде конденсатора, аккумулятора. По этому я использую слово "диск" для определения HDD, SSD и Flash-Drive, но не "накопитель". SSD диск в смысле цельного устройства хранения данных. Непонимания со стороны пользователй и коллег не встречал.

 Блок питания - предназначен для подачи напряжения комплектующим. Является "сердцем" компьютера. Однажды купив БП фирмы Gigabyte - ODIN 800 я отдаю предпочтения только модульным БП. Неиспользуемые провода затрудняют доступ к комплектующим. Критери выбора БП:
 - модульность
 - мощность
 - производитель
 - производимый шум

 Форм-фактор зависит от выбранного форм-фатора корпуса. Мощность расчитывается от потребляемой мощности компьютера и в моём случае главным потребителем напряжения будут пять дисков. Ниже изображён комплект поставки Thermartake M750W, который я выбрал с запасом, если и в дальнейшем буду добавлять комплектующие.
 - БП и проводами для подачи напряжения для процессора и материской платы
 - сумочка для хранения проводов
 - провода для подключения SATA, IDE, FDD, PCI устройств
 - кабель для подключения к сети 110\220 V.

 Сравнение двух БП. БП слева - ОЕМ БП для компании Packard Bell. Являясь БП для форм-фактора ATX может использоваться только в корпусах от Packard Bell. Ответ на вопрос "Почему?" чуть ниже.

 А вот и ответ: вентилятор. Он не позволит установить данных БП в большенство стандартных ATX корпусов, у которых есть "подставка" для БП. Обычно, это корпуса где БП устанавливаются в верхней части. ОЕМ корпус от Packard Bell имеет изменение для установки данного БП. Тем не менее, я использую этот БП в ремонтных работах.

 Видеокарта - предназначена для вывода видео (в некоторых случаях аудио) информации на монитор. Для выполнения работ, не связанных с обработкой графики можно использовать видео встроенное в процессор. К сожалению, встроенное в процессор видео тяжело справляется с flash анимацией... И простая flаsh-игра легко нагревает мой процессор i5-2540M в ноутбуке до 90 градусов. Критерии выбора видео карты:
 - производитель (у нас Unity программисты работают с Nvidia, я сам же предпочитаю ATI)
 - количество подключаемых мониторов
 - объём и тип видеопамяти
 - частота видео памяти
 - тип охлаждения
 - цена

 Ниже изображены несколько видеокарт. По мимо технических характеристик видеокарты отличаются по типу подключения. Верхняя справа видеокарта - это PCI. Ниже - PCI-E x1. Эти две видеокарты являются Low-profile (низкопрофильными) для установки в Micro-ATX корпуса. Так же эти две видеокарты имеют пассивное охлаждение. Две другие видео карты подключаются через PCI-E x16, имеют активное воздушное охлаждение. Если видеокарта является очень производительной, то ей необходимо дополнительное питание - на фотографии выделено красной рамкой.

 Есть несколько ограничений подключения мониторов, например, нельзя подключить два монитора через HDMI и D-Sub. Однако существуют переходники с HDMI в DVI (и не только такие). Но я не пробовал такое подключение. Существует ещё несколько подключений мониторов: Display Port, Thunderbolt. Второй является универсальным и пока используется в компьютерах производтсва Apple. Ниже изображены следующие варианты подключений для имеющихся видеокарт:
 - до четырёх DVI мониторов и S-Video (используется для подключения некоторых телевизоров)
 - два монитра через D-Sub\DVI или DVI\HDMI
 - до двух мониторов DVI и S-Video

 А вот такой монитор используется нашими разработчиками для наблюдения за серверами.

 И часть компьютера с видеокартой с 6-ю Mini-HDMI портами.

 Оперативная память - энергозависимая память, предназначенная для хранения используемой в рабочий момент информации. Рекомендуется использовать память от одного производителя. При использовании памяти от разных производителей с разными чипами - память может отказаться работать друг с другом.  На моей практике с разными чипами память не работала. Память с разными частотами может использоваться и память с наибольшей частотой будет работать на наименьшей частоте одной из памяти. Критерии выбора оперативной памяти:
 - объём
 - частота
 - производитель

 Критерий таймингов важен для настройки производительности памяти. Мне достаточно заводских таймингов, по этому этот критерий я пропускаю. Ниже изображены следующие типы памяти (сверху вниз):
 - DDR3
 - DDR2 с пассивным охлаждением
 - DDR2
 - DDR1
 - DDR2 для ноутбука

 Ключи у планок не дадут возможность использовать память в слотах с неподдерживаемым типом. Существуют материнские платы с разными слотами для подключения разной памяти. Большой объём памяти нужен для работы большего количества виртуальных машин, для некоторых программ обработки графики.
