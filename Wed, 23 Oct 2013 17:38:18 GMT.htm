С появлением поддержки UEFI в Hyper-V 2012 R2 можно продемонстрировать установку ОС на диск GPT. Сам процесс установки отличается от установки на диск MBR только разбивкой диска. Установка проводилась на Hyper-V 2012 R2. Конфигурация хоста: Intel i7 2600 3.4 GHz, 16 GB RAM, 2000 GB HDD. Для гостя было выделено 2 GB RAM. Вставляем установочный диск и вперёд:

 Выбираем язык установки, формат времени и метод ввода.

 Выбираем редакцию установки и принимаем лицензионное соглашение.

 Выбираем тип установки: обновление на месте или чистая установка.

 Выбираем меню "Новый". Имеется ввиду раздел. Windows автоматически разобъёт ваш диск на нужные разделы.

 При установке на диск GPT создаётся три допонительных разделов. Если вам кто-то говорит, что системные разделы не обязательны, то вы можете прислушаться к этому мнению, только если оно исходит от разработчика Windows. Хотя, на самом деле Windows может работать без раздела Recovery. Без разделов System & MSR - не пробовал.

 Взглянем на диск с помощью утилиты командной строки DISKPART. Командную строку можно вызвать с помощью сочетания клавишь Shift+F10. Переключение между командной строкой и графическим интерфейсом установщика можно выполнять с помощью комбинации ALT+TAB. На рисунке показан листинг дисков и детализация выбранного диска.

 Что будет, если ваш диск является MBR ? Удалим все разделы в графическом интерфейсе и конвертируем ваш диск из GPT в MBR. Внимание! Операция конвертации удаляет все ваши данные на диске!

 В графическом интерфейсе пробуем создать раздел на диске MBR... Это займёт чуть больше времени, т.к. Windows автоматически конфертирует диск в GPT. Ещё раз обращаю ваше внимание: если на вашем диске был раздел с вашими данным - он будет удалён! Проверяем диск утилитой DISKPART - его тип действительно GPT.

 Продолжаем установку. По завершении задаём пароль для учётной записи Администратора.

 Параллельно с установкой Windows Server 2012 R2 на второй виртуальной машине я запустил установку Windows Server 2008 R2. Новая ОС установилась на 15 минут быстрее.

 После установки ОС не забудьте сменить имя компьютера и включить автоматическое обновление. Не считая обновлений для IE, сразу установится три обновления для ОС объёмом около 250 MB.
