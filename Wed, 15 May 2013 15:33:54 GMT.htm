Однако, возможны сравнения с предыдущими операционными системами (например, с Windows Server 2003). Здесь не будет статей о Windows Server 2008, т.к. из-за родства с Vista я не работал и не работаю с данной ОС. Статьи в основном предназначены для начинающих администраторов. Это ещё одна причина не освящать Windows Server 2008. Новичкам будет сложнее управлять Windows Backup из командной сроки, а именно только от туда можно сделать резервную копию system state. В Windows Server 2008 R2 это уже возможно и с помощью графического интерфейса. Разница, как говрится, на лицо.
