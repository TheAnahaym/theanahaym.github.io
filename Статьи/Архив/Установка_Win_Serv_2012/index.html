 Обзор операционной системы Windows Server 2012 начну с установки. Процесс установки будет в картинках. Установка проводилась на Hyper-V 2012. Конфигурация хоста: Intel i7 2600 3.4 GHz, 16 GB RAM, 2000 GB HDD. Для гостя было выделено 2 GB RAM. В данной конфигурации установка длилась 13 минут. И ещё 3 минуты после ввода пароля Администратора. Microsoft решили переименовать Windows Server 8 Beta на Windows Server 2012. Что, на мой взгляд, правильно. Не раз встречал темы про Windows 8 в разделах для серверов... Установка Windows Server 2012 мало чем отличается от установки Windows Server 2012 RC. Краткое описание в картинках:

 

 

 Windows Server 2012 доступен только в двух редакциях. При установке только ядра (Server Core Installation) в дальнейшем можно вернуть графический интерфейс без переустановки ОС.

 

 

 

 

Небольшое отступление от темы. На следующем рисунке изображена разбивка диска хоста с Hyper-V ролью и RAID-0. Установка была на EFI. Пока не ясно, почему на диске, учавствующем в RAID-0, появился раздел напоминающий MSR. Ведь диски то MBR, а не GPT, как загрузочный диск.

 

 

 

 

 

 После установки необходимо переименовать* сервер и установить все обновления.

 

* Скрин сделан с сервера настроенного как контроллер домена.
